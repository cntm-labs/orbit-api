# Architecture Directory

This directory stores all documentation regarding the project's structure, tools, open-source dependencies, database schemas, and overall system architecture.

## Suggested Contents:
*   `project_structure.md`: Detailed breakdown of the repository and code modules.
*   `database_schema.md` or ERDs: Documentation of the PostgreSQL tables and relationships.
*   `tech_stack.md`: Information on Spring Boot, Java 25, PostgreSQL, and other dependencies.
*   `integrations.md`: Details about external APIs (e.g., crypto price feeds, payment gateways).
