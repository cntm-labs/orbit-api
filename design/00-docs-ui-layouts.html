Users API
Manage user registration and retrieval via Clerk identity synchronization.
POST /api/v1/users Register a new user
Request Body
Field Type Required Description
clerkUserId string required External Clerk user identifier
email string required User email address (format: email)
firstName string optional User's first name
baseCurrency string optional Default: THB
Example Request
cURL
JavaScript
Python
curl -X POST http://localhost:8080/api/v1/users \
  -H "Content-Type: application/json" \
  -d '{"clerkUserId": "user_abc", "email": "beam@orbit.app"}'
Response · 200 OK
200 Success
{
  "success": true,
  "message": "User registered",
  "data": {
    "id": "550e8400-e29b-41d4-a716...",
    "email": "beam@orbit.app",
    "status": "ACTIVE"
  }
}
GET /api/v1/users/clerk/{clerkUserId} Get user by Clerk ID

Parameter Type Required Description
