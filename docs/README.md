# Orbit API Documentation Site

This is the interactive API documentation site for the Orbit Personal Finance Management System, built with **Next.js**, **Tailwind CSS**, and **daisyUI**. It dynamically renders the OpenAPI (Swagger) specifications generated by the Spring Boot backend.

## Architecture

*   **Framework:** Next.js (App Router, Static Export)
*   **Styling:** Tailwind CSS + daisyUI
*   **API Rendering:** `swagger-ui-react`
*   **Package Manager:** Bun
*   **Hosting:** GitHub Pages (Automated via CI/CD)

## Local Development

To run the documentation site locally, you first need the backend's `openapi.json`.

### 1. Start the Backend API
Navigate to the root of the project and start the Spring Boot application:
```bash
./mvnw spring-boot:run
```
*(Ensure it is running on `http://localhost:8080`)*

### 2. Download the OpenAPI Spec
Open a new terminal, navigate to the `docs` directory, and fetch the latest spec from your local backend:
```bash
cd docs
curl -s http://localhost:8080/v3/api-docs > public/openapi.json
```

### 3. Run the Next.js App
Still inside the `docs` directory, start the Bun development server:
```bash
bun install
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the interactive documentation.

## Deployment

This site is automatically exported as a static HTML/JS bundle and deployed to GitHub Pages via the `.github/workflows/docs-and-tests-cd.yml` workflow every time code is merged to the `main` branch.
